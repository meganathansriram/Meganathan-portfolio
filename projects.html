Key Projects

Dimensionality Reduction Project

Designed and implemented a novel k-means clustering approach for dimensionality reduction. Improved activity recognition accuracy by 0.73% using a Gaussian Naive Bayes model and cut training time by 0.017 seconds through efficient feature selection.

Vehicle Routing Optimization with Genetic Algorithms

Developed an optimized solution for the vehicle routing problem using DEAP-based genetic algorithms. Created custom fitness functions to minimize travel distance and time, resulting in a 20% improvement in solution quality. Simulated delivery scenarios with 25 unique customer locations on a 100×100 grid.

Driver Fatigue Detection System

Built a real-time driver fatigue detection system using CNN and Haar Cascade classifiers. Leveraged in-vehicle camera data to detect facial expressions and eye movements associated with fatigue, improving road safety by delivering timely alerts.

MediScan – Intelligent Disease Prediction

Created a multi-disease prediction platform (Diabetes, Heart Disease, Breast Cancer) using Random Forest and ANN models. Implemented automated preprocessing and feature scaling with an interactive Gradio interface providing real-time predictions and confidence scores.
